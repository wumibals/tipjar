# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.x     | ✓         |

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Send details to the maintainers with:

- A clear description of the vulnerability
- Steps to reproduce
- Potential impact

You should receive a response within 72 hours. Confirmed issues will be patched as soon as possible.
